Detailed Description

template<typename Derived>
class EOSCRTP< Derived >

Base class for equation of state (EOS) implementations using CRTP.

The EOSCRTP class template provides a common interface for EOS implementations. Derived classes must implement the PressureSoSImpl method, which computes pressure and speed of sound based on density, internal energy and EOS-specific parameters.
