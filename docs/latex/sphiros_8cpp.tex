\doxysubsection{Detailed Description}
Main entry point for the SPHiros application. 

This file initializes the MPI and Kokkos runtimes, parses command-\/line arguments, and demonstrates the usage of various EOS (Equation of State) implementations. It also includes examples of using Kokkos views and performing computations with EOS objects.
