\doxysubsection{Detailed Description}
Implements the stiffened gas equation of state. 

The {\ttfamily \doxylink{classEOSStiffenedGas}{EOSStiffened\+Gas}} class computes pressure and speed of sound for a stiffened gas model. It inherits from the {\ttfamily \doxylink{classEOSCRTP}{EOSCRTP}} base class and provides the implementation for the {\ttfamily Pressure\+So\+SImpl} method.
