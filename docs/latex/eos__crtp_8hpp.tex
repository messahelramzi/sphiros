\doxysubsection{Detailed Description}
Defines the CRTP (Curiously Recurring Template Pattern) base class for equation of state (EOS) implementations. 

This file provides the {\ttfamily \doxylink{classEOSCRTP}{EOSCRTP}} class template, which serves as a base class for equation of state (EOS) implementations. It uses CRTP to delegate functionality to derived classes without the overhead of virtual functions in order to be GPU-\/compliant.
